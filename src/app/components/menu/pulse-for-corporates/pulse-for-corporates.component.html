User Accounts

Corporate user accounts are separate to personal accounts. An employee can have both a Pulse personal account and a Pulse corporate account. The registration process and user profile settings includes additional identifying fields for corporate employee accounts.

Once a corporate entity has signed up for a subscription, the email domain is activated. Any user using that domain as their email address in the signup process is automatically identified as a corporate user. The user profile will present additional data fields for completion.
